Neovim Keybindings

<leader> = Space

Общие

Клавиша Действие
<leader>w Сохранить файл
<leader>q Закрыть
<leader>a Выделить всё
<leader>j Добавить строку ниже
<leader>k Добавить строку выше
<leader>o Новая строка ниже, курсор туда
<leader>O Новая строка выше, курсор туда
<leader>; Добавить ; в конец строки
<leader>- Добавить пробел после курсора
<leader>_ Добавить пробел перед курсором
<leader>s Source текущего файла
<leader><CR> Перенести остаток строки на следующую
<leader><Tab> Вставить Tab в normal mode
<leader>t Открыть терминал внизу
<leader>h Закрыть другие окна + открыть Vifm
gb Назад (jump list)
n / N Следующий/предыдущий результат поиска (по центру)
* / # Поиск слова под курсором вперёд/назад (по центру)
g* / g# Как */#, без границ слова
<C-u> / <C-d> Страница вверх/вниз (по центру)
<C-h/j/k/l> Переключение между окнами
<C-S-h/l> Уменьшить/увеличить ширину окна
<C-S-j/k> Уменьшить/увеличить высоту окна
<M-k> / <M-j> Переместить строку вверх/вниз
